# medievAIl_bAIttle_generAIl

A medieval battle simulation with terminal-based visualization.

## Setup

1. Create a virtual environment:

   ```bash
   python -m venv venv
   ```

2. Activate the virtual environment:
   - On Linux/Mac: `source venv/bin/activate`
   - On Windows: `venv\Scripts\activate`

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the terminal view:

```bash
python View/terminal_view.py
```

Controls:

- P: Pause/Resume
- +/-: Adjust tick speed
- M: Toggle zoom
- ZQSD: Move camera
- A: Toggle auto-follow
- F: Toggle UI
- Ctrl+D: Toggle debug view
- Tab: Generate HTML report
- ESC: Quit
